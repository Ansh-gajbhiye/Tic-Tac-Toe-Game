## Features
- Interactive game board with hover effects
- Game history tracking (move through past moves)
- Win detection with highlight
- Responsive design works on mobile and desktop
- Clean, modern UI

## Technologies Used
- React
- JavaScript (ES6+)
- CSS3
- HTML5
